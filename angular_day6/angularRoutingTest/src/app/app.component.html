Angular Routing used to develop single page applications. For Example: Gmail App where all the contents of specific tab(Primary/Social/Promotions/Updates/Forums) displayed in the same page.

About SPA : The single-page application (SPA) basically a web app whose all functionalities are framed on a single page where users can navigate different sections of the page by clicking on its dedicated links or buttons.

Feature of SPA : 1. Simple to use interface. The biggest advantage of creating a SPA is retaining the user’s focus on the web app. The user doesn’t need to navigate between pages hence the user experience is improved 2. No Page Refresh. In SPA, the resources don’t need to get refreshed every time a user clicks on a different division of the page. 3. Code is rendered on the same page. SPA is the best when it comes to code optimization as all the HTML, CSS, and JS files are rendered on the same page.
